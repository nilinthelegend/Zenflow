# Project Rules & Customizations

This file outlines workspace-scoped rules and style guides for this project.

## Image Generation Rule
* **No AI Images:** Never run AI image generation tools unless the user explicitly requests one or provides specific permission first. This applies to website visual templates, project cards, previews, user interface screenshots, mockups, or any other visual asset.
* **Request Real Assets:** If a visual layout asset or preview image is required for a project card, prompt the user to provide the real screenshot or file path rather than generating a substitute.
